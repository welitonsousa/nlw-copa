#### runner server
this is a [Node](https://nodejs.org/) project

<b>node >= 18.22.0</b>

```js
cd server               // enter the directory
docker-compose up -d    // up local database 
npx prisma migrate dev  // apply changes database
npm install
npm run dev
```
